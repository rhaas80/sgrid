# Ian Hinder's NRAR tool:
-------------------------
The NRARAnalysis code and a README are available at
https://bitbucket.org/ianhinder/nraranalysis
The code processes NR data in the Numerical Relativity Data Format into
radially-extrapolated strain waveforms along with error estimates, as used
in the NRAR paper.
